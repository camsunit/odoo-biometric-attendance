Biometric Integration Module wth Odoo HR Attendance

This module integrates the various biometric attendnce machines with odoo server directly (without using the intermediate desktop computer). The the attedance/punch data gets pushed to the HR Attendance module on real time basis from the biometric device. It can also be customized to receive the user and biometric information from hardware and add through restful api from the odoo server application

Feature Highlights

  • All the cams biometrics machines listed at Cams Products do support this integration module
  • 90% of zkteco machines that support ADMS or WDMS or Web-Server features, are supported by this module. To confirm if your zkteco machine can be supported, verify at Developer Bot for Biometric Web API
  • And most of the biometric machines from eSSL, BioMax, Identix, and various brands available in across the globe are supported by our API, provided they are verified at Developer Bot for Biometric Web API
  • It will not use any manufacturer's software comes along with the biometric attendance machine. The communication happens through API, hence, you only need the internet connection to your biometric device. Punches will be recorded in Odoo on real time. Any punches recorded offline, will be pushed to the odoo once the biometric device comes online.
